Request objects
Name Type Value
method String Required
url String (IRI) Required

The method field is an uppercase String with one of: GET, HEAD, PATCH, POST, PUT, DELETE, etc. The url field is an IRI (or, if desired one of: URI, URL, URN).

Name Type Value
httpVersion String Required in 1.2, Optional in 2.0
_httpVersionNumber Integer Optional

The httpVersion field is a String containing the HTTP version string, which is usually verbatim the same as the string sent and received in the case of HTTP/1.0 and HTTP/1.1, or "HTTP/2" in the case of HTTP/2. The matching information may also be found in the _httpVersionNumber field, which must be 10, 11, or 20 respectively, which can be caluclated by 10*(major) + (minor) where the version string is "HTTP/(major).(minor)". If the minor version number is missing, it is assumed to be zero.
