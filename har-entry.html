startedDateTime DateTime Required
time Number Required

The startedDateTime field indicates the instant in which the connection was requested. The format of this DateTime must be in full ISO 8601 format, i.e.: %Y-%m-%dT%H:%M:%SZ (C/C++), yyyy-MM-dd'T'HH:mm:ss'Z' (Java), and must be expressed in the UTC timezone. The time field represents the total elapsed time in milliseconds (ms). For more information about time-related fields, see Timings objects.

Name Type Value
request Request Required
response Response Required

The request field holds a Request object, which represents a request sent from the client to the server. The response field holds a Response object, which represents a response sent from the server to the client.
